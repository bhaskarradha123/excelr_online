Css Positions:

    1.static
    2.relative
    3.absolute
    4.fixed
    5.sticky


1.static:
    position: static;
    -default value
    -top,right,left,bottom are not set

2.relative:
    position: relative;
    -relative to its normal positions
    -top,right,left,bottom are set
    -other elements are not affected by this element
    -can be used to move an element from its normal position
